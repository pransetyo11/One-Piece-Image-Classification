# One Piece Character Image Classification

## Deskripsi Proyek

Proyek ini merupakan implementasi Deep Learning untuk klasifikasi gambar karakter anime One Piece menggunakan Convolutional Neural Network (CNN). Model dilatih untuk mengenali berbagai karakter utama dari anime One Piece berdasarkan gambar.

Dataset yang digunakan berasal dari Kaggle dengan total 11.737 gambar yang terbagi ke dalam 18 kelas karakter.

Dataset digunakan dalam kondisi asli dengan resolusi yang tidak seragam serta memiliki variasi orientasi gambar, seperti rotasi atau perubahan warna pada beberapa gambar.

## Dataset

Dataset yang digunakan pada proyek ini berasal dari Kaggle.

Link dataset:
https://www.kaggle.com/datasets/ibrahimserouis99/one-piece-image-classifier

Jumlah data: 11.737 gambar
Jumlah kelas: 18 kelas karakter

Karakteristik dataset:
- Resolusi gambar tidak seragam
- Terdapat variasi orientasi gambar
- Beberapa gambar mengalami rotasi atau perubahan warna
- Dataset digunakan dalam kondisi asli tanpa standarisasi resolusi pada data sumber

## Metode

Model dibangun menggunakan pendekatan Deep Learning dengan Convolutional Neural Network (CNN) untuk melakukan ekstraksi fitur dari gambar karakter.

Tahapan utama dalam proyek ini meliputi:
1. Pengumpulan dataset
2. Pembagian dataset menjadi data training, validation, dan testing
3. Pembuatan model CNN
4. Proses training model
5. Evaluasi performa model
6. Konversi model 
7. Inference


## Hasil Model

Model dilatih untuk mengklasifikasikan gambar karakter One Piece ke dalam 18 kategori berdasarkan fitur visual yang dipelajari dari dataset.

Evaluasi model dilakukan menggunakan beberapa metrik seperti accuracy dan loss untuk melihat performa model selama proses training dan validation.

## Tujuan Proyek

Tujuan dari proyek ini adalah membangun sistem klasifikasi gambar karakter One Piece menggunakan metode Deep Learning berbasis Convolutional Neural Network.

Proyek ini juga dibuat sebagai submission akhir atau proyek akhir pada kelas Belajar Fundamental Deep Learning yang diselenggarakan oleh Dicoding Indonesia.
